Запуск программы 
Build --> Debug --> kurs.exe

В поле ниже надписи "Выберите тип уравнения" нужно выбрать нужный вам тип уравнения
После того, как тип уравнения выбран, вводите коэффициенты при A,B и(или) C
После того, как выбран нужный тип уравнения и все требуемые коэффициенты введены, нажимаете кнопку "Выполнить расчет" и увас появляется окно с решениями уравнения
